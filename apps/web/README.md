# Amanah Quran Web Placeholder

This directory is reserved for the future Amanah Quran web app.

Do not implement web features during Android V1.

The future web app must follow the same rules as the Android app:

- No ads.
- No analytics.
- No tracking.
- No login by default.
- No cloud sync.
- No donation popups.
- Verified Quran content only.
- Content source attribution and validation must remain visible.
