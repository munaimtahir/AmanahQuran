# Non-ADB Page/Juz Functional Checklist

1. Open Quran navigation.
2. Open Surah list.
3. Open Juz list.
4. Open Page list.
5. Open Page 1.
6. Open last page.
7. Open Juz 1.
8. Open Juz 30.
9. Switch IndoPak/Uthmani on page reader.
10. Bookmark an ayah.
11. Bookmark a page.
12. Open page bookmark.
13. Search `2:255` and open result.
14. Continue Reading opens correct location.
15. Enable Elder Mode and inspect page list/page reader.
16. Test Light/Dark/Sepia in page reader.
17. Confirm no login/network/permission prompt.
18. Mark ADB/device verification as pending.
