# Offline Functional Checklist

Run with airplane mode enabled:

1. Open app.
2. Open Surah list.
3. Open Surah reader.
4. Switch script.
5. Save last-read.
6. Reopen Continue Reading.
7. Add bookmark.
8. Remove bookmark.
9. Open Bookmarks screen.
10. Search Surah.
11. Search ayah reference.
12. Search Arabic text.
13. Open Trust Center.
14. Change theme.
15. Enable Elder Mode.
16. Disable Elder Mode.
17. Confirm no login, network, or permission prompt.

Do not claim device validation complete unless it has been performed on a real device.
