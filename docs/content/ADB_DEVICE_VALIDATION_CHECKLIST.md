# ADB Device Validation Checklist

1. Build debug APK.
2. Install APK on physical device using ADB.
3. Enable airplane mode.
4. Open app.
5. Open Quran navigation.
6. Open Surah list.
7. Open Juz list.
8. Open Page list.
9. Open Page 1.
10. Open Page 2.
11. Open last page.
12. Open Juz 1.
13. Open Juz 30.
14. Switch IndoPak/Uthmani on page reader.
15. Bookmark an ayah.
16. Bookmark a page.
17. Open page bookmark.
18. Search `2:255` and open result.
19. Continue Reading opens correct location.
20. Enable Elder Mode and repeat Page/Juz checks.
21. Test Light/Dark/Sepia in Page/Juz reader.
22. Confirm no internet required.
23. Confirm no permission prompt.
24. Confirm no login prompt.
25. Collect logcat only if a crash occurs.
26. Record device model, Android version, APK variant, pass/fail notes.
