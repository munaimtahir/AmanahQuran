# Amanah Quran AI Development Pack

This folder contains AI-agent-facing development instructions.

It does not replace the main project documentation in `/docs`. It adds implementation guardrails, scaffolding guidance, sprint prompts, data contracts, testing gates, release gates, and future web-app readiness notes.

## Use Order

1. Read `/AGENTS.md`.
2. Read all existing files in `/docs`.
3. Read this folder.
4. Run project discovery.
5. Create or update only the requested scaffold/code.
6. Report changed files and tests.

## Important

V1 remains Android-first and Sacred Reader only. The repository should be organized so a future web app can be added later without disrupting the Android app.
