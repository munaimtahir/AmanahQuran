# 09 — Do Not Build Yet

The following are explicitly **not allowed** in V1.

## Content Features Not Allowed

- Audio recitation.
- Urdu translation.
- English translation.
- Tafsir.
- Word-by-word meaning.
- Root/lemma study.
- Notes.
- Complex collections.
- Hifz tools.
- Tajweed correction.
- Recitation recording.
- AI assistant.

## Islamic Utility Features Not Allowed

- Prayer times.
- Qibla.
- Islamic calendar.
- Hadith database.
- Masjid directory.
- Zakat calculator.

## Platform Features Not Allowed

- Login.
- User accounts.
- Cloud sync.
- Server backend.
- Web app implementation.
- Push notifications.
- Social sharing feed.
- Public profiles.

## Monetization Not Allowed

- Ads.
- In-app purchases.
- Subscriptions.
- Donation popups during reading.
- Sponsored links.
- Affiliate links.

## Tracking Not Allowed

- Analytics SDK.
- Ad SDK.
- Advertising ID.
- Device fingerprinting.
- Location tracking.
- Background tracking.
- User behavior collection.

## Why

V1 must prove trust first. It must be a polished, verified, offline Sacred Reader before any expansion.
