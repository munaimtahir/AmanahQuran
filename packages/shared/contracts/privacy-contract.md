# Privacy Contract

## Principles
1. **No Tracking:** No analytics SDKs, no event tracking, no user behavior monitoring.
2. **No Ads:** No advertisement SDKs, no ad banners, no promotional popups.
3. **No Personal Data Collection:** No login, no accounts, no storage of user identifiers.
4. **Offline-First:** All core features must work without an internet connection.
5. **Transparency:** Source attribution and validation status must be clearly visible to the user (Trust Center).

## Technical Constraints
- No INTERNET permission for core functionality.
- Local storage only (no cloud sync in V1).
- No third-party SDKs that require network access or track users.
